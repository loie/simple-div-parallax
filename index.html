How it works

This beautiful animation is made possible by some HTML, CSS, nice graphics and a very simple jQuery script. Just read on to know how.

This demo works in current releases of Chrome, Firefox, Opera and Safari.

The Html

The HTML part is simple and pretty straightforward: The first part are the links. Just assign a unique id to each link. It is not necessary to put them into a single div.

<div>
  <a href="#" id="eins">First</a>
  <a href="#" id="zwei">Second</a>
  <a href="#" id="drei">Third</a>
  <a href="#" id="vier">Fourth</a>
</div>
<div id="scene">
    <div id="sky"></div>
    <div id="forest"></div>
</div>

Graphics

First of all, you need three graphics. The one for the non-moving background (i.e. the moon) you need an image as big as the div (i.e. 1000x500 pixels). The second image must be a little broader than the div. This example uses a dimension of 1200x500 pixels. For transparency effects it's recommended to use PNG graphics with non-opaque colors. The third image will be placed frontmost and has to be very broad. This example uses an image with 3000x500 pixels.

The Css

Set up the scene with the surrounding div width 1000x500 pixels. The scene has a background image which has the same dimensions as the scene itself. The other divs have an individual background image each. The z-index is used to order the layers in the correct manner. 

#scene {
    height: 500px;
    width: 1000px;
    
    background-image: url(../img/moon.jpg);
    background-position: 0 0;
    background-repeat: no-repeat;
}

#scene > div {
    height: 500px;
    width: 1000px;
    background-position: 0 0;
    background-repeat: no-repeat;
}

#sky {
    background-image: url(../img/sky.png);
    z-index: 1;
}

#forest {
    margin-top: -500px;
    background-image: url(../img/forest.png);
    z-index: 2;
}

The jQuery Part

This is where the action happens. The basic idea is to move the background of the individual divs after clicking on a navigation link. The position one represents the background position 0px 0px. animate provides a smooth transitions from one numeric value to another. $(selector) selects the element on which the animation will be applied. {backgroundPosition: '-667px 0px'} animates the CSS Property "background-position" to -667px 0px. 600 means the animation will take 600ms. swing sets a nice easing to the animation. Linear transitions are just too boring! ;-) 

$(document).ready(function(){
    $('#one').click(function(event) {
        event.preventDefault();
        $('#sky').animate({backgroundPosition: '0px 0px'}, 600, 'swing');
        $('#forest').animate({backgroundPosition: '0px 0px'}, 600, 'swing');
    });

    $('#two').click(function(event) {
        event.preventDefault();
        $('#sky').animate({backgroundPosition: '-66px 0px'}, 600, 'swing');
        $('#forest').animate({backgroundPosition: '-667px 0px'}, 600, 'swing');
    });

    $('#three').click(function(event) {
        event.preventDefault();
        $('#sky').animate({backgroundPosition: '-133px 0px'}, 600, 'swing');
        $('#forest').animate({backgroundPosition: '-1333px 0px'}, 600, 'swing');
    });

    $('#four').click(function(event) {
        event.preventDefault();
        $('#sky').animate({backgroundPosition: '-200px 0px'}, 600, 'swing');
        $('#forest').animate({backgroundPosition: '-2000px 0px'}, 600, 'swing');
    });
});
